# Aspecto - Deploy Vercel

Site estatico da Aspecto Solucoes Digitais.

## Publicar manualmente

1. Acesse https://vercel.com/new
2. Escolha "Deploy manually" ou arraste esta pasta para a area de upload.
3. Framework preset: Other.
4. Build command: deixe vazio.
5. Output directory: deixe vazio ou use `./`.
6. Clique em Deploy.

Depois do deploy, a Vercel gera um link `.vercel.app`.
